# CLI

My personal collection of golang cli utils

Initialized with the [Cobra CLI](https://github.com/spf13/cobra-cli/blob/main/README.md)

## Installation

```bash
go install github.com/metruzanca/mc@latest
# or locally
go install
```

## Uninstall

```bash
# Bash
rm $(go env GOPATH)/bin/mc
# Fish
rm (go env GOPATH)/bin/mc
```
